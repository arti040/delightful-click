Delightful Click v. 1.1

Amazingly awesome button on-click effects for your app!

  • Lightweight - less than 4KB minified!
  • No dependencies - only jQuery's required.
  • Support for Velocity animation engine.
  • Easy customizable - even a monkey can write an effect!

Examples

Download

Go to my Github profile, where you can find developer as well as minified version of the plugin.

Integration

Simple as fuck. Just attach a plugin script. You can do this however you want. Below just an example:

<script src="/path/to/the/script/jquery.delightful-click.min.js"></script>

Options

List of available options:

effect: 'available: wave, waveSquare, fade, fillUp or fillDown', //default: wave
color: 'valid CSS color', //default: #ddd
speed: 'speed in milliseconds', //default: 400
engine: 'animate or velocity', //default: animate
effectClass: 'class of an effect element' //default: del-effect
